# UNIX CLI Tutorial

A comprehensive, hands-on tutorial covering UNIX command line fundamentals: introduction, files & directories, permissions & links, text processing, process & job control, and shell scripting.

## Modules
- 01-intro-cli
- 02-files-directories
- 03-permissions-links
- 04-text-processing
- 05-process-job-control
- 06-shell-scripting
- quick-reference

## Quick start
Clone the repository and explore each module:

```bash
git clone https://github.com/bianglalametro/unix-cli-tutorial.git
cd unix-cli-tutorial
```

Each module contains:
- Explanations and examples
- Practice exercises
- Runnable scripts
