### The Linux Command Line
**Author:** William Shotts  
**Price:** Free (CC licensed)  
**Link:** [linuxcommand.org](http://linuxcommand.org/tlcl.php)

A comprehensive introduction to the command line. Covers everything from basic navigation to shell scripting. Available as a free PDF download.

**Topics covered:**
- Shell basics
- Navigation and file manipulation
- Redirection and pipelines
- Expansion and quoting
- Permissions
- Shell scripting

---

### Linux Pocket Guide
**Author:** Daniel J. Barrett  
**Publisher:** O'Reilly Media

A quick reference guide that fits in your pocket. Perfect for looking up commands while working.

**Best for:**
- Quick reference
- Command options lookup
- Essential daily commands

---

### Learning the Bash Shell
**Authors:** Cameron Newham, Bill Rosenblatt  
**Publisher:** O'Reilly Media

A thorough introduction to Bash shell. Covers both interactive use and scripting.

**Topics covered:**
- Command-line editing
- Job control
- Shell programming
- Customization

## ðŸŸ¡ Intermediate Level

### Classic Shell Scripting
**Authors:** Arnold Robbins, Nelson H.F. Beebe  
**Publisher:** O'Reilly Media

Master shell scripting with this comprehensive guide. Covers portable shell programming that works across different UNIX systems.

**Topics covered:**
- Text processing with sed and awk
- Regular expressions
- Portable scripting
- Best practices

---

### UNIX and Linux System Administration Handbook
**Authors:** Evi Nemeth, Garth Snyder, Trent R. Hein, Ben Whaley  
**Publisher:** Addison-Wesley

The definitive guide for system administrators. Covers both UNIX and Linux in depth.

**Topics covered:**
- System administration fundamentals
- Networking
- Security
- Automation
- Cloud computing

---

### Linux Bible
**Author:** Christopher Negus  
**Publisher:** Wiley

A comprehensive guide covering all aspects of Linux. Updated regularly with new editions.

**Topics covered:**
- Installation and configuration
- Desktop and server Linux
- Administration
- Security
- Networking

---

## ðŸ”´ Advanced Level

### Advanced Programming in the UNIX Environment
**Author:** W. Richard Stevens, Stephen A. Rago  
**Publisher:** Addison-Wesley

The classic reference for UNIX system programming. Essential for understanding how the system works at a deep level.

**Topics covered:**
- File I/O
- Process control
- Signals
- Interprocess communication
- Network programming

---

### The UNIX Programming Environment
**Authors:** Brian W. Kernighan, Rob Pike  
**Publisher:** Prentice Hall

Written by UNIX pioneers, this classic explains the philosophy behind UNIX tools and how to use them effectively.

**Topics covered:**
- UNIX philosophy
- Shell programming
- Document preparation
- Text processing

---

### sed & awk
**Authors:** Dale Dougherty, Arnold Robbins  
**Publisher:** O'Reilly Media

The definitive guide to two of the most powerful text processing tools in UNIX.

**Topics covered:**
- Regular expressions
- sed commands and scripts
- awk programming
- Advanced techniques

---

### Effective awk Programming
**Author:** Arnold Robbins  
**Publisher:** O'Reilly Media

A comprehensive guide to awk, the pattern scanning and processing language.

**Topics covered:**
- Language fundamentals
- Regular expressions
- One-liners and scripts
- Advanced features
