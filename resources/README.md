## üéØ Learning Paths

### Path 1: Casual User
For everyday command-line use:
1. Complete [01-Introduction](../01-introduction/) and [02-Files and Directories](../02-files-and-directories/)
2. Practice with [Beginner Exercises](../exercises/beginner/)
3. Keep the [Essential Commands Cheatsheet](../cheatsheets/essential-commands.md) handy

### Path 2: Developer
For software development workflows:
1. Complete all core modules (01-06)
2. Focus on [Text Processing](../04-text-processing/) and [Shell Scripting](../06-shell-scripting/)
3. Practice with [Intermediate Exercises](../exercises/intermediate/)
4. Learn Git command-line (external resource)

### Path 3: System Administrator
For system administration tasks:
1. Complete all modules
2. Deep dive into [Processes and Jobs](../05-processes-and-jobs/) and [Advanced Topics](../07-advanced-topics/)
3. Complete all exercises including [Advanced](../exercises/advanced/)
4. Study additional resources on networking and security

### Path 4: DevOps Engineer
For automation and deployment:
1. Complete all modules with focus on scripting
2. Master [Shell Scripting](../06-shell-scripting/)
3. Complete all exercises
4. Study CI/CD, containers, and infrastructure as code

## üé§ Interview Preparation

### Common Interview Topics

1. **File System Navigation**
   - Explain absolute vs relative paths
   - Describe the Linux file system hierarchy

2. **Permissions**
   - Explain chmod 755 vs 644
   - When would you use setuid/setgid?
   - What's the sticky bit?

3. **Process Management**
   - Difference between `kill` and `kill -9`
   - How to find processes consuming resources?
   - Explain zombie processes

4. **Text Processing**
   - When to use grep vs sed vs awk?
   - How to count occurrences in a file?
   - How to extract specific columns from CSV?

5. **Scripting**
   - Explain `set -e` and `set -o pipefail`
   - How to handle errors in scripts?
   - What's the difference between `$@` and `$*`?

### Practice Questions

See each module's `exercises.md` for topic-specific questions.
